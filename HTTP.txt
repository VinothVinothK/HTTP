HTTP 1.1 vs HTTP 2

HTTP stand for Hypertext Transfer Protocol. 
Its supports on all web browsers. 
HTTP is send and receive information between computer or client and server.
Http is the client and server model.

HTTP 1.1

It loads on browser one by one only. While loading one file all others files are cannot be load.
Here a server only serves content to a client device if the client asks for it.

HTTP 2
Its Load multiple resource data at single time. So its loading time is faster than the http1.1
In Http 2 by allowing a server to “push” content to a client before the client asks for it.
Http2 a more advanced compression method called HPACK that eliminate a more redundant information in http header packets.
